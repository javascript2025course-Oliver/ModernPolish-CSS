# ModernPolish CSS Framework - Features

## 1. Reset & Base

* Full margin/padding reset
* Base fonts, colors, and line-height

## 2. Layout Utilities

* `.container` for max-width content
* `.flex`, `.flex-center`, `.flex-column` for flexible layouts
* `.grid`, `.grid-cols-2`, `.grid-cols-3`, `.grid-cols-4` for grid layouts
* Responsive behavior for mobile devices

## 3. Typography

* Headings: `h1` to `h6`
* Paragraphs `p` and small text `small`

## 4. Buttons

* `.btn` for standard buttons
* `.btn-outline` for outlined buttons
* Smooth hover effects

## 5. Cards

* `.card` with hover lift effect
* Rounded corners and shadowing

## 6. Forms

* Styled `input`, `textarea`, `select`
* Focus highlights
* `label` styling

## 7. Alerts

* `.alert` base class
* Variants: `.alert-success`, `.alert-error`, `.alert-warning`, `.alert-info`

## 8. Navbar

* `.navbar` container
* Link hover effects
* Responsive layout for mobile

## 9. Utility Classes

* Margin: `.mt-1-3`, `.mb-1-3`
* Padding: `.p-1-3`
* Text alignment: `.text-center`, `.text-right`, `.text-left`
* Rounded corners: `.rounded`
* Shadows: `.shadow`, `.shadow-lg`
* Background colors: `.bg-white`, `.bg-light`, `.bg-dark`

## 10. Animations & Transitions

* `.fade-in` animation
* `.scale-up` hover effect

## 11. Responsive Design

* Grid and flex layouts adapt for mobile
* Navbar adjusts to column layout on smaller screens
